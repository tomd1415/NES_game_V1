FamiStudio quick start

  1. Install FamiStudio (free, runs on Windows / macOS / Linux).
  2. Compose a song.
  3. File → Export → FamiStudio Sound Engine assembly, format ca65.
  4. Drag the .s file onto this page (or click the upload buttons below).

Want to hear something straight away? Click Load starter pack on the right — you can replace it with your own work whenever you like. Full guide in AUDIO_GUIDE.md.

🎼 Songs

Each song is a FamiStudio-exported .s file. The one marked default plays when the game starts.

💥 Sound effects

A single FamiStudio sfx-export file holds up to 128 sound effects. Each "Song" in your FamiStudio sfx project becomes one slot here.
