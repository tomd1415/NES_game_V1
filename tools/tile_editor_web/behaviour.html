Behaviour editor — quick help

What this page does. You pick a behaviour type (ground, wall, platform, door, trigger, or one of your custom slots) and paint it onto background tiles. Each sprite then has a reaction for each type — land, block, bounce, exit, ignore, or call_handler.

Keyboard

  • 0…7 — pick behaviour type
  • P pencil · F fill · R rect · E eraser
  • Ctrl+Z undo · Ctrl+Shift+Z redo
  • Shift+click with pencil picks the type under the cursor

Nothing you do here changes the tile graphics — just which cells count as ground / wall / etc. The Play-in-NES pipeline reads this map (Phase B) to apply the reactions.
