About auto-save

Your work is saved to this browser's storage every time you change anything. Every 30 seconds a numbered snapshot is kept (last 5). Every 5 minutes an emergency JSON backup is offered for download. If anything goes wrong, hit Recover… in the toolbar to load any snapshot.

Recover from a snapshot

Pick any snapshot below and click Restore. The current project is first saved to its own snapshot slot so nothing is lost.
