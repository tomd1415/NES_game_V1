  compilable main.c.  The rest of the pipeline —
  compiling with cc65 and running in the emulator — is
  exactly the same as the Code page.

  
    
    
  1. Make sure at least one sprite is tagged Player
      on the Sprites page.
    2. 
    
  2. On the Builder page, tick the modules you want.  Fill in
      the settings (start position, walk speed, jump height, …).
    3. 
    
  3. Watch the Problems panel on the right — it
      lists anything that would stop your game from working.
    4. 
    
  4. When it's all green, press ▶ Play.
